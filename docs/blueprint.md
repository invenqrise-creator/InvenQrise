# **App Name**: InvenQrise

## Core Features:

- Product Catalog: Browse and manage product details like name, images, and pricing.
- Category Management: Create, edit, and delete product categories to organize the catalog.
- Basic Inventory Tracking: Track product stock levels using virtual locators or sub-inventories (e.g., front-of-house, back-of-house).
- Low Stock Alerts: Receive automated alerts when product stock falls below a defined threshold.
- Sales Data Dashboard: Visualize sales trends and top-selling products with an interactive dashboard.
- Customer Insights Tool: Suggest the best category to add a given product to. This AI powered tool helps in refining customer targeting based on past purchase data and preferences.
- Marketing Campaign Interface: Create basic marketing campaigns for a specific category, including selecting the date that they run, and selecting the items to which it applies.
- Roles: Admin, Stock Keeper, Inventory Manager, Marketing Manager.
- Role-specific dashboards: Role-specific dashboards.
- Password hashing, token refresh, role-based access control: Password hashing, token refresh, role-based access control.
- User Management: User Management

## Style Guidelines:

- Primary color: Fresh Mint Green (#98FF98) to represent growth and freshness, reflecting the supermarket's offerings.
- Background color: Very Light Beige (#F5F5DC) for a clean and neutral backdrop.
- Accent color: Light Orange (#FFB347) to highlight important actions and draw attention to marketing campaigns.
- Headline font: 'Belleza', sans-serif with personality aligned to fashion, art, and design.
- Body font: 'Alegreya', serif, contemporary.
- Use clean, modern icons for product categories and actions.
- Maintain a clean and organized layout to easily showcase products and promotions.
